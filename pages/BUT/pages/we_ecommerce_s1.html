Application web : SAE1.04

Description du projet

Pour ce projet nous devions partir du besoin d'un client et réaliser une base de données pour satisfaire ce besoin après un entretien.
La première étape du projet était de réaliser un rapport d'analyse des besoins qui regroupe une présentation de l'entreprise (client) et ses besoins, une synthèse des données pertinentes utiles pour l'entreprise, un schéma du MCD et une justification des choix de la modélisation choisie.

Ensuite nous devions coder, donc écrire sur un script SQL permettant de tester notre application et enfin utiliser ce script et l'implémenter dans une application web (en Flask Python)
